# vagrant-centos

This envirment is CentOS7 use vagrant.  
You can launch the foundation of the web application in one word "vagrant up".  

```
vagrant up
-> http://192.168.33.101/index.php
```

* OS
    * CentOS7
    * centos/7
* Web Server
    * Apache2.4.6
* PHP
    * PHP5.4.16
* Database
    * MySQL5.6
